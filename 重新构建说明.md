##  问题原因

### 之前的错误
```
fatal: destination path '/Users/runner/theos/sdks' already exists and is not an empty directory.
Error: Process completed with exit code 128.
```

**原因：** GitHub Actions runner 上已存在 SDK 目录，导致 git clone 失败。

### 解决方案

修改了 `.github/workflows/build.yml`，在克隆 SDK 前先清理已存在的目录：

```yaml
- name: Download iOS SDK
  run: |
    # 先清理已存在的目录
    rm -rf $HOME/theos/sdks
    git clone --recursive https://github.com/theos/sdks $HOME/theos/sdks
```

## 📱 安装指南（构建完成后）

### 方式 1: SSH 安装（推荐）

```bash
# 1. 下载的 deb 文件
# 2. 上传到设备
scp packages/DYYY_*.deb root@你的设备 IP:/tmp/

# 3. SSH 连接并安装
ssh root@你的设备 IP
dpkg -i /tmp/DYYY_*.deb
respring
```

### 方式 2: Filza 安装

1. 将 .deb 文件传到设备 `/var/root/`
2. 打开 Filza App
3. 点击 .deb 文件 → 安装
4. Respring

### 方式 3: Sileo/Cydia 本地源
